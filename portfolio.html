Technical Projects

Neural Network Library

Spring 2020 at College Station - (CNN Link) (FNN Link)

I built a convolutional and feed forward neural network libary using Python
to classify RGB images with customizable layered network architectures. The library
was implemented via backpropagation algorithms and contains support for calibrating deep
learning hyperparemeters, regularizers, and optimizers. The convolutional neural network
built from this library achieved a greater than 99% validation accuracy on the benchmark
70,000 MNIST handwritten images database.

Teachable App

Spring 2021 at Dallas - (Teachable Link)

I made a peer to peer messenger application for students and teachers to directly interact
with each other over coursework and academic matters in Android Studios. I implemented front-end
tasks using XML to generate the application UI and Kotlin to interact with the UI elements for in-app
functionalities. The back-end consisted of the Google Firebase API to store and retrieve user information
in cloud hosted JSON trees.

Siamese Based Metric Learning Model

Summer 2021 at Dallas - (Publication Link)

As an undergraduate machine learning researcher under Dr. Theodora Chaspari in the Texas A&M Human Bio-Behavioral
Signals AI Lab, I had the fortune to publish an academic paper on using siamese based metric learning to classify
meal macronutrient content from postprandial glucose signals. Our metric learning approach performed better than
previous models, benefitting nutrition measurement and diabetes management in particular. Our paper was accepted into
the prestigious IEEE Biomedical and Health Informatics conference in July 2021, where I had the honor to present my paper
in front of distinguished faculty and researchers around the world.

The Social App

Fall 2021 at College Station - (TheSocialApp Link)

For HowdyHack 2021, our team created a mobile Android application for young people to conviniently find
and create social events. The Social App serves as an effective alternative to using social media or direct
messaging to attend/organize events, which can be cumbersome and limited in reach. In addition to incorporating
a social events feed, we also utilized the Google Map API to create a map view for the user to easily see all the nearby
events. The app is currently still under construction.
